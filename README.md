## Website

Live site: https://aibersemi.my.id/

## Development Lokal

Jalankan server lokal:

```bash
./server.sh --run
```

Website lokal akan tersedia di:

```text
http://localhost:8099
```

Hentikan server:

```bash
./server.sh --stop
```
