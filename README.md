AiBersemi IT professional portfolio website.

This repository is used as a work log and development archive for a personal website. The code is open so it can be viewed, studied, or downloaded, but this repository is not prepared as a public contribution project.

## Website

Live site: https://aibersemi.my.id/

## Local Development

Start the local server:

```bash
./server.sh --run
```

The local website will be available at:

```text
http://localhost:8099
```

Stop the server:

```bash
./server.sh --stop
```
